## Features

- Full Material Design 3.0
- Dynamic Theme Colors Scheme
- MobX state management

## Development

Clone the repository, then install the dependencies and generate .g files:

    flutter pub get
    flutter pub run build_runner watch

Start the app:

    flutter run
